Minecraft 1.16.5 原版辅助mod整合

发表于2022-02-17更新于2022-02-17

该MOD整合主要面向1.16.5原版生存玩家，所有MOD均可支持原版服务器使用

MOD均为fabric端MOD，请确认的客户端已经安装fabric

mod介绍

Advanced Compass 在画面上方提供一个生物雷达，在其中显示附近的生物，可在mod菜单中进行设置

AdvancementInfo 为进度页面添加详细信息，例如完成进度所需的条件

Amecs 引入了几个修饰键（alt、control 和 shift），以便在进行键绑定时提供更多选择，同时在控制页面添加搜索栏功能

AntiGhost 反幽灵方块，通过按“G”键（可在设置中更改），或输入/ghost指令激活

AppleSkin 在HUD添加饱和度和疲劳度的可视化，在持有食物时恢复的潜在健康状况的可视化，在工具提示中添加食物价值信息

ASH-Another Simple HUD 轻量化信息显示，可在mod菜单中进行设置

AuroraKeystrokes 显示在键盘上按下了哪些键，同时显示了CPS(注：不统计客户端内连点器，定时点击等mod的使用情况）。通过输入/keystrokes访问的配置GUI

Auth Me 在客户端内部而不是启动器更改玩家ID及游戏内正版登录，可在mod菜单中进行设置

Auto Config GitHub项目wiki

Auto Reconnect 自动重新连接服务器

Better Dropped items 更好的掉落物渲染

Better Enchanted Books 为附魔书添加颜色以更好地区分辨别附魔书类型，可在mod菜单中进行设置

Better Ping Display 以ms精确显示玩家延迟

Better Sodium Video Settings Button 更好的钠模组设置按钮（汉化）

Better Third Person 更好的第三人称，类似越肩视角

BetterF3 更好的F3信息显示，，可在mod菜单中进行设置

BiomeUtils GitHub项目链接

Bounding Box Outline Reloaded 简称BBOR,可在画面中显示结构（例如女巫小屋，海底神殿）及出生点范围，可在“控制”中进行设置按键绑定

Carpet carpet熟肉讲解 carpet GitHub项目链接

Chat Heads 在聊天栏显示玩家头颅

Chat up 将聊天框位置上移以避免遮挡盔甲GUI

ChayEmotes 快捷颜文字输入，通过/emote激活，可在配置文件中进行更改

Cherished Worlds 添加收藏存档功能，被收藏的存档将会被置顶

Chuck Random Reversal 更简洁（相比于f3）的显示区块边界同时显示区块中心等信息

Click Through 可以与带有告示牌，展示框的物品（容器，音符盒等）直接交互，潜行时点击来旋转物品展示框中的物品（注：该功能可能会导致在部分推人式刷冰机中玩家无法利用展示框交换即将损坏的镐子）

Client Commands Earthcomputer写的mod功能整合，可以快捷计算玩家随机数，RNG附魔等功能相关链接1 相关链接2 相关链接3

Color Unchained 提供彩色字体等支持

Custom Crosshair 自定义准星，可在“控制”中进行设置按键绑定设置GUI

Custom Selection Box 自定义选择框,展示了一个清晰简洁的选择框，可以根据自己的喜好进行设置。通过指令/csbconfig激活设置GUI 相关链接

Dark Logding Screen 自定义更改客户端启动加载界面颜色等，可在mod菜单中进行设置

DCCH 保存聊天历史

Durability 101 为物品添加耐久显示

Dynamic FPS 在离开游戏时降低FPS以降低不必要的GPU负载

Dynamic Sound Filters 更好的音效

Easier Villager Trading 快捷村民交易，可以直接点击交易栏完成交易

Egg Tab 将生物蛋和附魔书作为单独的选项页面

Elytra Auto Flight 飞行时在画面左上角显示飞行状态（高度，速度等），通过激活热键自动在飞行时调整视角实现不使用火箭达到持续飞行状态，可在“控制”中进行设置按键绑定

Emotexraft 更多动作，通过激活热键做出动作

Enhanced Attack Indicator 显示更多操作的攻击指示器进度，例如进食，投掷物品等，可在mod菜单中进行设置

Entity Outliner 高亮实体，通过U键开启GUI(注：部分服务器（尤其是pvp类服务器）使用该mod将会被视为作弊行为）

Extra Player Renderer 额外玩家渲染，通过f8开启GUI设置 相关链接

Fabric API 部分mod前置，建议安装

Facric Zero 处理部分mod不兼容现象，同时优化游戏卡顿 相关链接

Gatewayer 空手右键末地折跃门/返回门显示相关信息

Held Item Info 显示手持物品（工具）信息

Hydrogen 氦 元素周期表系列mod之一，内存优化

Inventory HUD 实时显示自身药水效果，背包物品信息，盔甲耐久度等，可在mod菜单中进行设置

Inventory Profiles 快捷整理背包（r键整理），并提供部分实用性功能（例如禁止受到伤害摇晃画面等），可在mod菜单中打开设置GUI

Inventory Tabs 在不退出背包或容器GUI的情况下快速访问玩家周围可以访问的容器（或其他方块，例如工作台）

InvMove 在玩家开启GUI时仍可继续移动，可在mod菜单中进行设置

Iris 光影渲染器，可兼容Sodium等mod（内置Sodium，无需额外安装）相比于OptiFine拥有更强的光影优化，（美中不足的是无法设置光影）

Item Scroller 提供一系列鼠标手势操作，以及快捷合成功能 相关链接

Litematica 投影mod，用于辅助建造。masa全家桶之一 相关链接

Lithium 锂 元素周期表系列mod之一，游戏运算优化，可显著降低CPU负载

MaLiLib masa全家桶mod前置

microDurability 实时显示盔甲耐久

MiniHUD 显示客户端及服务端信息等（例如fps，tps,mspt），同时可用于可视化显示地形，结构范围等（类似BBOR) 相关链接

mod Menu mod菜单，部分mod前置，建议安装

MultiConnect 跨版本链接服务端（类似VIA)，可以连接低版本服务端

Not Enough Crashes 阻止客户端崩溃并在崩溃时显示崩溃原因，崩溃时不需要重启客户端

Not Enough Creativity 创造模式扩展，在创造模式下进入生存模式物品栏右侧“→”进入该模式 相关链接

Oh My Minecraft Client 提供一些实用性功能（例如高亮岩浆源等），可在mod菜单中进行设置

Orderly 简洁的生物生命值显示，可在mod菜单中进行设置

Pet Owner 显示被驯服的生物的主人玩家ID

Phosphor 磷 元素周期表系列mod之一，优化光照运算

seed Cracker 存档seed反推 相关链接

SignCopy 蹲下右键告示牌复制其内容

Smooth Boot 降低客户端启动加载时CPU负载

Smooth Scrolling Everywhere 使原版GUI列表滚动变得平滑

Sneak Tweak 平滑的潜行动画，可在mod菜单中进入设置GUI更改动画速度或禁用潜行动画

Sodium 钠 元素周期表系列mod之一，提供史诗级客户端FPS优化，iris内置（如果你安装了内置钠的iris，那你无需再安装Sodium

Time To Live 显示tnt和creeper的爆炸倒计时

tweakeroo 万能工具箱，提供大量实用性功能，masa全家桶之一 相关链接

Untranslated Items 将物品所显示的名称改为英文

VoxelMap 小地图，提供作为一个小地图它该有的功能（废话），可在“控制”中进行设置按键绑定以打开设置GUI

WI Zoom 提供类似OptiFine的望远镜功能，滑动鼠标滚轮进行再放大或缩小，可在“控制”中进行设置按键绑定

Xaero’s Minimap（汉化） 小地图，提供作为一个小地图它该有的功能（废话），可在“控制”中进行设置按键绑定以打开设置GUI

与VoxelMap不同的是，Xaero’s Minimap拥有更多且更强的功能设置(美中不足的是没有像VoxelMap那样的全屏大地图）

友情提醒

部分mod功能可能会违反服务器规定，请确认或得到允许后使用
